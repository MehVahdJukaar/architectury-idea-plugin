Reports methods annotated with @VirtualOverride that are not valid virtual overrides.

This inspection verifies that annotated methods correctly correspond to a virtual override target. If a method is marked with @VirtualOverride but does not match the expected signature or cannot be resolved as a virtual override, an error is reported.

This helps prevent incorrect annotation usage and ensures that platform-specific override mechanisms are applied only where valid.

Example: 


// incorrect usage
@VirtualOverride("fabric")
public void doSomething(int x) {
    // no matching virtual method
}

// correct usage
@VirtualOverride("fabric")
public void doSomething() {
    // matches virtual override target
}

Ensure that annotated methods match the expected signature and exist in the virtual dispatch system.
